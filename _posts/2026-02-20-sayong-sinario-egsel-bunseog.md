사용 시나리오 엑셀 분석
    + 1. 로그 분석
        + 너무 다양한 상황이 존재할 수 있어 안하는 방향으로 검토
    + 2. 로그 분석
        + 마찬가지로 너무 다양한 상황이 존재할 수 있어 안하는 방향으로 검토중
        + 문제는 일종의 이상탐지를 해야한다는 점임
        + 정상적인 상황을 어떻게 정의할 것인가? 하는 문제가 있음
        + 이런 지식들을 어떻게 저장하는게 좋을까?
        + 벡터 디비를 사용하는게 좋을까? 등의 고민이 있음
    + 3. SSL 분석
        + 제시된 시나리오에 대해서 처리하는 것은 어려움이 없어 보임
        + Gemdefine 분석(33, 35, 37 메세지 분석)
            + 37 메세지를 분석해서 해당 CEID가enable되어 있는지 확인
            + 35 메세지를 분석해서 CEID가 어떤 리포트에 정의되어 있는지 확인한다.
            + 35 메세지는 어떤 이벤트 발생시 어떤 리포트가 발행되는지를 정의 하고 있음
            + 연결된 리포트 발행시 어떤 VID들이 보고되고 있는지 확인한다.
    + 4. 성능 모니터링
        + CPU메모리 사용량 저먹ㅁ
        + 사용량은 특정 DB에 기록되고 있음. 이 DB를조회해서 사용량 보고서를 만들 수 있다.
    + 5. SSL 시나리오 도식화
        + XML 을읽어서 그림 그려 주도록 하면 됨
        + 만약 실 시나리오를 원한다면 이것은 가능하지 않음. 단순 SSL파싱 및 도식화라면 가능함
    + 6. PM 후 점검
        + 점검항목에 대해 자동 점검 수행
        + 로그를 입력으로 정해진 항목들을 검사하면 됨
    + 7. 문의 대응
        + 큐브 채팅 기록을 토대로 일종의 RAG된 출력을 생성해 주면 되겠으나 문제는 큐브 채팅 기록에 의미있는 정보가 별로 없음
    + 8. 로그추출
        + 특정 장비의 GEM LOG추출 요청
        + H log, 또는 앱 로그에서 GEM define항목을 추출 하면 되는데, 가장 최근 로그에 없는 경우 나올 때 까지 계속 로그를 뒤져야 하는 문제가 있음.
    + 9. 핑 로스 체크
        + SENET 로그 등을 가져와서 핑 로스를 확인하면 됨
    + 10. SSL 관리
        + 장비의 타입별로 SSL 타입이 다름. 그런데 이 SSL 파일의 유사도 등을 비교 하고 싶은 니즈가 있음. 파일만 비교하는 것은 아니고 DB를조회해서 해당 SSL 파일에 대한 여러 정보를 가져와서 비교할 수 있기를 원하고, 비슷한 SSL 파일을 찾아 오는 기능을 원함
